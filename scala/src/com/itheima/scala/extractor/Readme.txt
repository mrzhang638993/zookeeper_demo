#  下面使用自定义的泛型。java中的泛型定义存在很多的问题的。
# 泛型的上界和下界的定义。
泛型的上界：表示的是必须是这个类的本身或者是这个类的本身进行继承的，是需要这个类的子类的。
泛型的下界：必须是这个类的本身或者是这个类的父类的。
类型存在上界和下界的话，先写下界然后书写上界的。示例代码如下：
def  demo[T >:PoliceMan <:Person1](array: Array[T])=println(array)

1.scala中默认是非变：类型之间的关联关系在泛型中是不会存在体现的。没有任何的从属关系的
class Pair[T]{}
2.协变：泛型体现出来的关系和原有的继承关系一致。
class Pair[+T]
3.逆变：泛型体现出来的关系和原始的继承关系相反。参数化类型的关系和原始的继承关系相反
class Pair[-T]
