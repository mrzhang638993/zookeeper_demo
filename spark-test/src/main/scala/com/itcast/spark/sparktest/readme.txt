出租车的要点：
1.根据出租车上下坐标点的位置需要计算出来对应的行政区域的名称；
需要使用到geoJson处理坐标的？
2.需要区分每一个车辆的一次的阶段，包括载客开始和载客结束的时间点
需要进行session划分操作？
3.需要计算连续两次的载客过程中的间隔操作。也就是等待时间
需要确认是同一个人的连续的上车时间和下车时间的。
4.根据得到的区域和对应的每一次的出租的等到时间，得到区域的名称和平均的载客时间间隔的。
5.数据的过滤和清洗操作？设计到大量的ETL的过程和相关的实际操作实现的。
